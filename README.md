## Features

- **Current Weather:** Get up-to-the-minute weather data for any location, including temperature, humidity, wind speed, and conditions.
- **Location Search:** Easily find weather information for any city or region around the world.
- **Responsive Design:** Enjoy a seamless experience on desktop and mobile devices.

## Getting Started
To get your Weather App up and running, follow these steps:

### Prerequisites

Before you can run the Weather App, make sure you have the following prerequisites installed:

- [Node.js](https://nodejs.org/) - Ensure you have Node.js installed. You can check this by running `node -v` in your terminal.

### Usage
1. Open your web browser and navigate to http://localhost:3000 (or the URL where your app is hosted).
2. Use the search bar to find weather information for a specific location.
3. Explore the current weather and the 5-day forecast.
